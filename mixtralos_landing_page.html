TÍNH NĂNG
[DE]

Custom Desktop (PyQt6)

Toàn bộ giao diện viết từ đầu bằng Python + PyQt6: taskbar, launcher, splash screen, settings.

[RU]

MXL Script Engine

Ngôn ngữ script riêng mxl-run và mxl-builder tích hợp sẵn trong hệ thống.

[VG]

Tương thích VGA rộng

Hỗ trợ VESA, fbdev, Intel, AMD, NVIDIA basic — chạy tốt trong VirtualBox và VMware.

[NM]

NetworkManager

Kết nối WiFi/LAN tự động ngay khi boot, có nm-applet trên thanh taskbar.

[LV]

Live Boot

Chạy thẳng từ USB, không cần cài đặt. Dữ liệu không ghi vào ổ cứng máy chủ.

[UE]

BIOS + UEFI

ISO hybrid grub-mkrescue, hỗ trợ cả máy cũ (BIOS) lẫn máy mới (UEFI Secure Boot off).
